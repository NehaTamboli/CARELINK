##  Project Overview

CARELINK is a full-stack intelligent platform designed to support non-profit organizations by integrating **Agentic AI**, **Generative AI**, and **DevOps-driven infrastructure automation** into a unified system.

The platform acts as a bridge between users, administrators, and infrastructure by automating key processes such as **message classification, donation management, and deployment pipelines**. Using Infrastructure as Code (IaC), the system ensures that environments are reproducible, scalable, and free from manual configuration errors.

Additionally, CARELINK incorporates AI capabilities to analyze user messages, determine urgency levels, and generate meaningful responses. This combination of AI + DevOps transforms traditional systems into **smart, automated, and scalable solutions**.

---

##  Key Features (Detailed)

###  Agentic AI System
The system uses an agent-based AI approach to autonomously process incoming messages. It classifies messages into categories such as donation, emergency, or support and assigns urgency levels (High, Medium, Low). If the AI model is unavailable, the system seamlessly switches to a rule-based fallback, ensuring uninterrupted functionality.

---

###  Generative AI (GenAI)
CARELINK integrates Generative AI to produce intelligent and context-aware responses. This enables dynamic interaction with users, providing meaningful suggestions, responses, and assistance without manual intervention.

---

###  Quiz Tutor Chatbot
An interactive chatbot is included to assist users with queries and learning-based interactions. It supports real-time communication and enhances user engagement through AI-powered responses.

---

###  Dashboard UI
The system includes a modern and intuitive dashboard that allows administrators to:
- Monitor incoming messages
- Analyze trends and urgency levels
- Manage donations and user interactions
- View analytics and system performance

---

###  DevOps & Infrastructure Automation
CARELINK follows modern DevOps practices to automate the complete lifecycle of application deployment:
- Infrastructure is provisioned using Terraform
- Applications are containerized using Docker
- Kubernetes manages container orchestration and scaling
- CI/CD pipelines automate build, test, and deployment processes

This ensures consistency, scalability, and reliability across all environments.
