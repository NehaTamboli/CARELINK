# 🚀 CARELINK  
### AI-Powered Non-Profit Support System with DevOps Automation

---

## 📌 Project Overview

CARELINK is a full-stack intelligent platform designed to support non-profit organizations by combining **Agentic AI**, **Generative AI**, and **DevOps-based Infrastructure Automation**.

The system automates:
- Infrastructure provisioning (IaC)
- Application deployment (CI/CD)
- Intelligent message analysis (AI)

This ensures a **scalable, reliable, and production-ready system**.

---

## 🧠 Key Features

### 🤖 Agentic AI System
- Autonomous decision-making for message classification
- Identifies urgency (High / Medium / Low)
- Categorizes messages (Donation, Emergency, Support, etc.)
- Uses fallback rule-based system if AI is unavailable

---

### ✨ Generative AI (GenAI)
- AI-powered response generation
- Context-aware recommendations
- Smart interaction with users

---

### 💬 Quiz Tutor Chatbot
- Interactive chatbot interface
- Helps users with queries and learning
- Real-time responses using AI logic

---

### 📊 Dashboard UI
- Message Management
- Analytics & Monitoring
- Donation tracking
- Admin dashboard

---

### ⚙️ DevOps & Infrastructure Automation

- Infrastructure provisioning using **Terraform**
- Containerization using **Docker**
- Orchestration using **Kubernetes**
- CI/CD using **GitHub Actions / Jenkins**
